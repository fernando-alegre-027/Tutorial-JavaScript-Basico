005 Comentarios

Los comentarios de JavaScript se pueden usar para explicar el código de JavaScript y hacerlo más legible. También se pueden usar para evitar la ejecución, al probar código alternativo.

Comentarios de una sola línea

Los comentarios de una sola línea comienzan con //.
Cualquier texto entre // y el final de la línea será ignorado por JavaScript (no se ejecutará).

Los comentarios de varias líneas comienzan con /*y terminan con */.
Cualquier texto entre /*y */será ignorado por JavaScript.
