Operadores y operandos

Los números (en una operación aritmética) se llaman operandos.
La operación (a realizar entre los dos operandos) está definida por un operador.

El operador de suma ( + ) suma números:

El operador de resta ( - ) resta números.

El operador de multiplicación ( * ) multiplica números.

El operador de división ( / ) divide números.

El operador de módulo ( % ) devuelve el resto de la división.

En aritmética, la división de dos números enteros produce un cociente y un resto. En matemáticas, el resultado de una operación de módulo es el resto de una división aritmética.

Incrementando

El operador de incremento ( ++ ) incrementa los números.

Decreciente

El operador de decremento ( -- ) decrementa los números.

Operador ternario

Es un operador que permite realizar una sentencia condicional (es decir, un if-then-else) en una sola línea

Un operador ternario está formado tres argumentos (de ahí su nombre); o más concretamente, por cinco partes:

  • La condición que se puede o no cumplir.
  • Un signo de interrogación.
  • Valor si la condición es cierta.
  • Dos puntos.
  • Valor si la condición es falsa.
