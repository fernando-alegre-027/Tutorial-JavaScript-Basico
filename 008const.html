008 Const

La palabra const clave se introdujo en ES6 (2015) .
Las variables definidas con const no se pueden volver a declarar.
Las variables definidas con const no se pueden reasignar.
Las variables definidas con const tienen ámbito de bloque.

¿Cuándo usar const?

Como regla general, siempre declare una variable const a menos que sepa que el valor cambiará.

Usar const cuando declare:

  • Una nueva matriz
  • Un nuevo objeto
  • Una nueva función
  • Una nueva expresión regular

Objetos constantes y matrices
